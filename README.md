# cpp-go-odyssey

> 一位后端开发工程师的 C++ 与 Go 学习之旅

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Last Updated](https://img.shields.io/github/last-commit/AliothEpsilon/cpp-go-odyssey)](https://github.com/AliothEpsilon/cpp-go-odyssey/commits/main)

`cpp-go-odyssey` 是我作为后端开发工程师的个人技术成长记录。这不仅仅是一个代码仓库，更是我系统性地学习 C++ 和 Go 语言、夯实计算机科学基础、探索现代后端系统架构的**知识图谱**与**实践日志**。

我的目标是：**在扎实的理论基础上，通过持续的深度学习和动手实践，追求在 C++ 和 Go 后端开发领域的高水平**。这个旅程始于入职，但远不止于此。

---

## 🚀 目标与愿景

*   **加强理解：** 超越“会用”，深入理解 C++ (现代特性、内存管理、并发) 和 Go (Goroutine、Channel、内存模型、性能剖析) 的**底层机制与最佳实践**。
*   **夯实基础：** 系统性地复习和深化对**数据结构与算法、操作系统、计算机网络、数据库**等计算机科学核心知识的理解。
*   **掌握架构：** 学习并实践**微服务、分布式系统、高并发、高可用、高性能**的设计原则与模式。
*   **工程卓越：** 提升**代码质量、测试能力、可维护性、可观测性**和 DevOps 实践水平。
*   **持续成长：** 养成结构化学习和深度反思的习惯，将知识内化为能力。

> **为什么是 "Odyssey"？**  
> 这趟旅程如同奥德赛，充满挑战与未知。每一次对技术原理的追问，每一次动手实践的尝试，都是这段旅程中不可或缺的篇章。
